Step 1 — JSONBin

Status Store URL

Create a free bin at jsonbin.io/app/bins → New Bin, paste {"status":"idle"}, save, copy the bin URL (looks like https://api.jsonbin.io/v3/b/XXXXX).

Step 2 — Scriptable Script

Copy this into a new Scriptable script named QueuePlayer

The script is shown in the code block below. In Scriptable, tap + → paste → save as QueuePlayer.


Step 3 — Shortcut

Build the "Play AM Song" Shortcut

  • Open Shortcuts → New Shortcut → name it exactly Play AM Song
  • Add action: Receive [Text] input from [Share Sheet and Nowhere Else]
  • Add action: Find Music → filter: Name → Contains → [Shortcut Input]
  • Add action: Play Music → [Music] (from step 3)
  • Add action: Wait → 2 seconds (so Music.app has time to start)
  • Done — Scriptable handles the rest.
Step 4 — Share Sheet Shortcut

Build the "Add to QEUE" Shortcut

  • New Shortcut → name it Add to QEUE
  • Add action: Receive [URLs] input from [Share Sheet]
  • Add action: Open URLs → value: your web app URL + ?addsc=[Shortcut Input]
  • In Safari, save this web app to home screen first, then get its URL from Settings.

When you share a SoundCloud link to this shortcut, it opens the app and auto-adds the track.

Step 5 — Scriptable URL

Your Scriptable Trigger URL

Scriptable runs scripts via: scriptable:///run/QueuePlayer

The app uses this automatically. Make sure your script is named exactly QueuePlayer.
